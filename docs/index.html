SSLAndriodTest

Android 虚拟定位应用 · 百度地图 + FusedLocation Mock

Kotlin Jetpack Compose 百度地图 SDK 7.6.4 FusedLocation Mock

核心功能

🗺️

百度地图选点

集成百度地图 SDK，支持拖动选点、POI 城市搜索、地点收藏管理，默认预置苏州坐标。

📍

三 Provider 注入

同时 Mock GPS、Network、Fused 三个 Provider，覆盖所有定位路径，确保第三方 App 读到模拟位置。

🔧

Fused Mock 模式

通过 FusedLocationProviderClient.setMockMode(true) 直接注入融合定位，解决高德地图等 App 绕过 GPS Mock 的问题。

🛡️

Mock 标记清除

反射调用 setIsFromMockProvider(false)、setMock(false)，清除 mIsFromMockProvider 私有字段，绕过 App 的 Mock 检测。

⏱️

前台服务定时推送

ForegroundService 每 300ms 调用 tick() 重新注入坐标，App 切后台不中断，启动时 burst 5 次确保立即生效。

🔄

完整坐标转换

WGS-84 ↔ GCJ-02 ↔ BD-09 四步转换算法，确保百度地图显示与系统 GPS 注入坐标准确对应。

🔒

强制权限检查

启动时通过 AppOpsManager 检查模拟位置权限，未设置时弹出不可关闭弹窗引导用户配置。

📡

真实定位兜底

停止模拟后点击定位按钮，跳过 lastLocation 缓存强制请求新 GPS 位置，3 秒超时保护。

使用步骤

安装应用

下载并安装 SSLAndriodTest APK 到 Android 设备（minSdk 30）。

开启开发者选项

设置 → 关于手机 → 连续点击 MIUI 版本/版本号 7 次开启开发者选项。

设置模拟位置应用

开发者选项 → 选择模拟位置信息应用 → 选择「SSLAndriodTest」。应用启动时会强制提示此步骤。

选择目标位置

拖动地图将图钉对准目标位置，或使用搜索/收藏功能快速跳转。

开始模拟定位

点击「开始模拟定位」按钮，系统将通过 GPS + Network + Fused 三个 Provider 注入模拟坐标。

验证效果

打开高德地图、智租换电等第三方 App，确认位置已变更为模拟位置。
